# Transwire

A landing page concept for a fictional multi-currency account aimed at freelancers and remote workers.

## 🔗 Live Demo

[tito-osakwe.github.io/transwire](https://tito-osakwe.github.io/transwire/)

## 📖 About

Transwire is a fictional product — a concept I built to practice landing page design and frontend development. The brief: a marketing page for a multi-currency account aimed at freelancers, with a punk-leaning editorial design rather than the usual fintech gradient look.

## ✨ Features

- Punk-editorial design with grain texture, asymmetric layouts, and offset hard shadows
- Accessible modal dialog with focus management, Escape-to-close, and scroll-lock
- Animated scrolling marquee with pure CSS keyframes
- Fully responsive — tilted cards reset to a clean stack on mobile
- Respects `prefers-reduced-motion` for accessibility

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, transforms, keyframe animations, inline SVG textures)
- Vanilla JavaScript (for the modal logic)

No frameworks, no build tools.

## 💡 What I Practiced

- Designing with bold, opinionated color palettes (electric blue, hot pink, acid yellow, lime)
- Asymmetric grid layouts and rotation-based visual hierarchy
- Building accessible modal dialogs from scratch
- Editorial typography with `Instrument Serif`
